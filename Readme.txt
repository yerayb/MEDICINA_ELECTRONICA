------ MEDICINA ELECTRONICA ------

1. CONFIGURACIÓN DEL PROYECTO

- Abrir la carpeta src con un editor de código (por ejemplo Visual Studio Code)
- (OPCIONAL) Crear un entorno virtual
- Instalar las dependencias a través de:  pip install -r requirements.txt
- Ejecutar app.py: python3 app.py 
- Acceder a la dirección: http://127.0.0.1:5000/

2. UTILIZARLO COMO ADMINISTRADOR

- Introducimos las credenciales de administrador
  Usuario: admin
  Contraseña: adminDesarrollo

- Ver medicos: Nos enseña una tabla con los medicos registrados en el sistema
- Agregar medico: Nos permite agregar un profesional medico al sistema, automáticamente generara un usuario para dicho profesional medico.
- Modificar medico: Modificar datos de profesionales medicos ya existentes en el sistema.
- Borrar medico: Borrar un profesional medico en el sistema. 

- Ver pacientes: Nos enseña una tabla con los pacientes registrados en el sistema
- Agregar paciente: Nos permite agregar un paciente en el sistema.
- Modificar paciente: Modificar datos de pacientes ya existentes en el sistema.
- Borrar paciente: Borrar un paciente en el sistema. 

3. UTILIZARLO COMO MEDICO
- 
- Si es el primer acceso, introducir solo el usuario para generar una contraseña nueva.

- Ver prescripciones: Permite al profesional medico ver las prescripciones medicas existentes en el sistema.
- Agregar prescripción: Permite al profesional medico agregar una prescripción medica a un paciente registrado en el sistema.
- Borrar prescripciones: Borrar una prescripción medica en el sistema.
